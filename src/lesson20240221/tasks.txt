1. Дан массив некоторых упорядоченных данных. Необходимо "честно" перемешать данные,
т.е. распределить их так, чтобы вероятность нахождения любого элемента на любом месте была бы одинакова.
Например, задача перемешивания колоды карт.
2. Создать класс Rectangle для хранения данных о прямоугольнике - ширина x, высота y.
Реализовать возможность добавления Rectangle в HashSet.
3. Создать для класса Rectangle компаратор по ширине x. Создать список прямоугольников и отсортировать его по ширине.
4. На основе Rectangle создать класс Square для хранения данных о квадрате.
Реализовать возможность добавления Square в HashSet и TreeSet.
