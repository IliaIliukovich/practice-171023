1. Установить VisualVM, запустить, проверить, как отображаются
- пользовательские потоки в режиме работы, сна, ожидания
- потоки-демоны
2. Запустить и посмотреть информацию о работе потоков в jstack
3. Спортивное соревнование. Бегут 3 бегуна. Каждый пробегает дистанцию за случайное время.
Каждый бегун земеряет свой результат и сообщает его тренеру. В конце забега выводится информация о победителе.
Реализовать данную модель с помощью запуска отдельных потоков. Бегуна представить через класс Runner implements Runnable.
